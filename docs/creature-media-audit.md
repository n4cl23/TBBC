# Creature media audit

Before this correction, all 34 records used the shared `/images/bestiary/beasts-hero.webp`. The preserved 1024×1024 PNG masters were matched by normalized filename and converted to optimized, slug-scoped WebP derivatives. No master was overwritten.

## Publication policy

- Hero, card and thumbnail must be readable WebP files under the creature slug.
- A realm placeholder is allowed only with an explicit “art in preparation” badge.
- A gallery reflects its real item count; a single render is never duplicated to simulate a set.
- A 3D viewer requires a published `.glb`, MIME `model/gltf-binary` and non-zero size. Source-only masters remain private.
