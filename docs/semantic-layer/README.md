# Camada semântica

Cada entidade recebe herança derivada de seu reino: cultura, paleta, linguagem material, afinidade, clima e identidade visual. `politicalAlignment` e `economicContext` permanecem `null` quando o Canon não apresenta evidência explícita.

`GraphService` fornece identidade, relações, vizinhança limitada, timeline, contexto para IA e busca indexada. `SemanticQueryService` responde governo, proteção, bosses, criaturas, artefatos, participantes, conexões, guildas e locais sem duplicar datasets.

O `EntityContextDTO` é o contrato preparado para futura integração com LLM: entidade, semântica herdada, relações, vizinhos, timeline e resumo factual.
